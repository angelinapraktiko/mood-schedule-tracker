# Mood & Schedule Tracker (Python)

## Deskripsi
Project Python sederhana untuk membantu pengguna mencatat jadwal kegiatan dan mood harian. Program ini dibuat menggunakan konsep Object-Oriented Programming (OOP) dan menyediakan fitur interaktif berbasis command line.

## Fitur Utama
- Menambahkan jadwal kegiatan harian
- Menampilkan seluruh jadwal
- Menampilkan jadwal berdasarkan hari tertentu
- Mencatat mood harian
- Validasi input mood
- Menampilkan rekap mood

## Teknologi yang Digunakan
- Python
- OOP (Class, Method, Encapsulation)
- Built-in library (datetime, collections)

## Cara Menjalankan
1. Pastikan Python sudah terinstall
2. Jalankan file utama:
   ```bash
   python main.py
