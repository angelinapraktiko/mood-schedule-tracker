# Mood & Schedule Tracker (Python)

Sistem sederhana berbasis Python untuk mengelola jadwal kegiatan dan mencatat mood harian menggunakan konsep Object-Oriented Programming (OOP).

---

## Deskripsi
Project ini merupakan aplikasi berbasis Command Line Interface (CLI) yang membantu pengguna dalam mencatat jadwal kegiatan serta memantau mood harian. Program dirancang menggunakan konsep OOP agar lebih terstruktur dan mudah dikembangkan.

---

## Fitur Utama
- Menambahkan jadwal kegiatan harian  
- Menampilkan seluruh jadwal  
- Menampilkan jadwal berdasarkan hari tertentu  
- Mencatat mood harian  
- Validasi input mood  
- Menampilkan rekap mood  

---

## Tools yang Digunakan
- Python  
- Object-Oriented Programming (Class, Method, Encapsulation)  
- Library: `datetime`, `collections`
